Bedrock 번역기 설정

AWS 콘솔 → Amazon Bedrock → API keys에서 발급한 키를 붙여넣으세요.
aws sts assume-role / SSO / 페더레이션으로 받은 임시 자격증명을 넣으세요. STS 임시 자격증명은 보통 12시간 내 만료되므로 만료 시 다시 발급해 갱신해야 합니다. 영구 자격증명이면 Session Token은 비워 두세요.
먼저 위 값을 저장한 뒤 아래 버튼을 누르면 새 탭에서 SSO 로그인 화면이 열립니다. 승인하면 임시 자격증명이 자동 발급됩니다 (aws sso login과 동일한 방식).
자격증명은 이 브라우저에만(chrome.storage.local) 저장되며 Bedrock 호출에만 사용됩니다.
서울 리전 고정입니다.
서울 리전은 apac. 추론 프로파일을 사용합니다.

API 키 발급 방법
1. AWS 콘솔에서 Amazon Bedrock 열기
2. 좌측 메뉴 API keys → Generate API key
3. Short-term(최대 12시간) 또는 Long-term 선택해 생성
4. 사용할 모델은 Bedrock → Model access에서 접근 권한이 활성화돼 있어야 합니다.
